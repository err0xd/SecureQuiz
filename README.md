# SecureQuiz
# 資通安全概論測驗（安全選擇題測驗系統 - 單檔版）

這是一個**純前端、單一 HTML 檔案**的選擇題測驗應用程式，專為安全考試設計。

### 主要特色
- **答案延後解密**：題目內容、選項、配分、章節為明文（考生可正常答題），但**正確答案與解析完全加密**，只有交卷時監考人員輸入 key 才能解密、計算分數並顯示解析。
- **防作弊**：考生答題過程中無法從瀏覽器 console 或原始碼看到正確答案。
- **單選 / 多選自動判斷**：根據答案數量自動切換 radio（單選）或 checkbox（多選），並顯示題型標示。
- **完整考試功能**：
  - 進度條（顯示已答、未答、註記）
  - 題目跳轉、註記功能
  - 交卷前提醒未答題目
  - 結果頁顯示總分、及格判斷、章節答對比例、詳細解析（正解綠色、誤選紅色刪除線）
- **管理者加密工具內建**：隱藏的管理者頁面，可從 CSV 產生明文題目 + 加密答案。
- **無需伺服器**：純靜態 HTML，直接開啟即可使用（離線也可）。

### 檔案說明
- 本專案只有一個檔案：`index.html`（或您自行命名）
- 使用 CryptoJS 進行 AES 加密（CDN 載入）

### 使用方式（考生）
1. 開啟 HTML 檔案
2. 點擊「開始考試」（無需輸入代碼）
3. 正常答題（支援單選 radio、多選 checkbox）
4. 點擊「交卷」
5. 監考人員輸入解密 key 後，即可看到得分與完整解析

### 使用方式（管理者設定題目）
1. 開啟 HTML 檔案
2. 點擊頁面右下角（或連結）「(管理者設定)」，輸入密碼 `admin` 進入加密頁面
3. 輸入及格分數
4. 貼上 CSV 格式題庫（每行一題）：
題號,題目內容,選項A,選項B,選項C,選項D,答案,解析,配分,章節
例如：
1,什麼是台灣的首都?,台北,高雄,台中,台南,A,台灣首都是台北,10,地理
2,以下哪些是水果?,蘋果,香蕉,胡蘿蔔,西瓜,AB,蘋果與香蕉是水果,15,常識

- 答案欄位：單選填單一字母（如 `A`），多選填多個（如 `AB` 或 `ABC`）
5. 輸入一組加密 key（自己記住，之後監考要用）
6. 點擊「產生加密資料」
7. 分別複製兩個輸出：
- 「明文題目資料」 → 貼到 HTML 原始碼的 `const plainQuiz = [...]`
- 「加密答案資料」 → 貼到 `const encryptedAnswers = ""` 內的雙引號中
8. 儲存 HTML 檔案，即完成題庫更新

> **重要**：更新題庫後，請妥善保管加密 key（監考時需要）。若遺失無法解密答案。

### 注意事項
- 本系統為純前端設計，適合小規模內部測驗或練習。
- 若用於正式重要考試，建議搭配後端伺服器進一步加強防作弊（例如答案比對移到伺服器）。
- CSV 格式若題目或選項內含逗號，請用雙引號包起來（目前簡易解析，可能需手動調整）。
- 支援離線使用（下載後斷網也可）。
